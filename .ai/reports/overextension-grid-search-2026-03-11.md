## 9. MDD 분석

### 9.1 MDD 발생 경위

MDD 8.18%는 **2026-02-26(피크) → 2026-03-04(바닥)** 4거래일간 발생:

```
02-26  54,722,218  +9.44%  10포지션  SIDEWAYS  ← 피크 (고점)
02-27  54,547,568  +9.10%  10포지션  SIDEWAYS
03-03  52,198,120  +4.40%  10포지션  BEAR       DD=4.6%
03-04  50,248,378  +0.50%  10포지션  STRONG_BEAR DD=8.2% ← 바닥
03-05  50,838,270  +1.68%  10포지션  SIDEWAYS    반등 시작
03-06  50,934,160  +1.87%  10포지션  SIDEWAYS    회복
```

### 9.2 원인

- 2월 말~3월 초 **시장 전체 급락** (SIDEWAYS → BEAR → STRONG_BEAR 2일 만에 3단계 하락)
- **10포지션 풀인 상태**에서 충격을 정면으로 맞음
- Exit Rules (stop loss 등)가 1일 만에 반응할 수 없는 급락 속도

### 9.3 Overextension Filter의 MDD 방어 효과

| 조건 | MDD |
|------|---:|
| Filter OFF | 8.18% |
| Filter ON | 8.32% |

**차이 +0.14%p → Overextension Filter는 MDD 방어에 기여하지 않음.**

이는 예상된 결과입니다. Overextension Filter는 **매수 진입 시점의 필터**이므로, 이미 보유 중인 포지션의 급락은 방어하지 못합니다. 급락 방어는 **Exit Rules (stop loss, trailing stop, profit floor)** 영역입니다.
